
# Meshy App MVP - Product Requirements Document

## 1. Product Vision
Meshy AI is a mobile-first platform designed to democratize 3D creation through advanced reconstruction. It enables artists, hobbyists, and engineers to transform simple images or text into production-ready 3D assets directly from their mobile devices.

## 2. Core Feature List

| Feature | Description |
| :--- | :--- |
| **Splash Screen** | High-fidelity entrance animation featuring the brand logo and core mission statement. |
| **Explore Feed** | Discovery engine for 3D models with themed channels, popularity stats, and interaction (likes). |
| **Image-to-3D** | Generation of 3D meshes from uploaded or captured images with credit-based cost. |
| **Text-to-Image** | AI image generation with optional reference image support for editing workflows. |
| **Task Management** | Real-time progress tracking for generation tasks with completion notifications and historical records. |
| **Interactive 3D Viewer** | Full-scale 3D preview environment with gesture hints, lighting adjustments, and asset specs. |
| **Export Matrix** | One-click export for professional formats (OBJ, STL, GLB, FBX, 3MF, etc.) with logic for paid access. |
| **3D Print Ecosystem** | Dedicated interface for hardware integration with partners like Creality and Bambu Handy. |
| **Subscription & Credits** | Tiered subscription plans (Free/Pro/Studio) and a credit top-up system with simulated Apple/Google Pay. |
| **Asset Library** | Categorized personal library for models, images, and exported files with management tools. |
| **Multi-Language Support** | Dynamic interface switching between English and Simplified Chinese for global reach. |
| **Auth & Security** | Secure login protocol featuring email OTP and third-party social integration mocks. |

## 3. UI/UX Specifications
- **Theme**: Sci-fi Dark Mode (Obsidian & Deep Space)
- **Primary Color**: #D0F870 (Acid Green)
- **Accent Color**: #C084FC (Neon Purple)
- **Typography**: Inter (Modern Casing - Title Case for UI)
- **Animations**: Spring-based transitions and glassmorphism overlays.

## 4. Monetization Strategy
- **Credit-Based Economy**: Actions cost Meshy Credits (25 for 3D, 5 for Image).
- **Subscription Tiers**: Pro and Studio tiers offer recurring credits, concurrent task slots, and higher priority.
